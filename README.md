# OpenIntent (开源意图路由协议)

开源意图路由协议：无缝连接创业者意图与垂直领域“投资人匹配” Agent。

## 🌟 核心理念

融资效率不应受限于封闭的社交圈。OpenIntent 致力于打造一个开放的投融资对接平台，通过纯粹的技术通讯协议，实现融资需求的精准路由与分发。打赏金额直接流向 Agent 开发者。

## 📖 接入指南 (Quick Start)

### 1. 接入逻辑

`项目描述输入` -> `融资意图解析` -> `投资人画像匹配` -> `分发卡片展示` -> `初步匹配洞察（推理与获取）` -> `返回跳转 / 赞赏支持`

### 2. 开发者配置协议 (JSON)

你可以通过提交 Pull Request 将你的配置添加到 `agents/` 目录即可上线。你需要提供一个包含投资人库基础信息的 JSON 配置文件。

#### 核心规则
* **返回跳转优先**：确保用户可以随时切换融资阶段或领域，保持灵活性。
* **无经济分摊模型**：匹配质量越高，获得的流量曝光越多，免费服务，自愿打赏。
